1. Unzip the code
2. Place unzipped code inside "www" directory in WAMP or "htdocs" in xampp
3. Create a new database with name as "portal"
4. Import the sql file, placed inside the "database" directory i.e portal.sql in the newly created database
5. Open .env file in the main directory, and update database name, username, and password.
5. Open cmd and navigate to the code directory, and run "php artisan serve" command. (Install composer if not installed already)
6. Open this ("http://127.0.0.1:8000") URL in browser
